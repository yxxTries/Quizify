=====================================
  HOW TO RUN QUIZAI
=====================================


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  FIRST TIME SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

You only need to do these steps once.

1. Install Ollama
   Download from: https://ollama.com
   Run the installer, then open a terminal and run:

       ollama pull llama3

   This downloads the AI model (~4 GB). Wait for it to finish.

2. Install Python
   Download from: https://python.org  (version 3.10 or newer)
   During install, check the box that says "Add Python to PATH"

3. Install Node.js
   Download from: https://nodejs.org  (version 18 or newer)

4. Install backend dependencies
   Open a terminal, navigate to the backend folder, and run:

       cd quiz-ai/backend
       pip install -r requirements.txt

5. Install frontend dependencies
   Open a terminal, navigate to the frontend folder, and run:

       cd quiz-ai/frontend
       npm install


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  RUNNING THE APP (EVERY TIME)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

You need 3 terminal windows open at the same time.
Or just double-click start.bat and it does all of this for you.

-- Terminal 1: Start Ollama --

    ollama serve

-- Terminal 2: Start the backend --

    cd quiz-ai/backend
    uvicorn main:app --reload

-- Terminal 3: Start the frontend --

    cd quiz-ai/frontend
    npm run dev

Then open your browser and go to:

    http://localhost:5173


To stop the app, press Ctrl+C in each terminal window.
